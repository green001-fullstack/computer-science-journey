# What Breadth-First Search (BFS) is

### Breadth-First Search (BFS) is a tree traversal algorithm that visits every node level by level before moving to the next level.

# How BFS differs from DFS.

### DFS explores one branch as deeply as possible before returning, while BFS explores all nodes at the current level before moving deeper.

# Why recursion isn't the natural choice for BFS.

### Recusrsion will only process the nodes along a particular path before going to the nodes alng other path.

# Why BFS uses a queue.

### Queues allowing processing of data that gets into it first before processing of the ones that enter them later(FIFO)

# Three real-world examples where BFS makes sense.

### Looking for an employee office in an organization building. You will check all the rooms in one floor first before moving to the next floor.
