# Root 

### The topmost node from which every other node originates.

# Parent 

### This is a node that is directly above another node 

# Child 

### A node directly below another node 

# Sibling 

### WHen nodes share the same parent 

# Leaf 

### A node with no children 

# Internal Node 

### A node with at least a child 

# Edge 

### A connection between two nodes 

# Path 

### A sequence of connected nodes 

# Ancestor 

### All the nodes that are above a particular node along the same path. 

# Descendant 

### All the nodes under a node along the same path.
