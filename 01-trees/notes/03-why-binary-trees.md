# What is a general tree? 

### A general tree is a tree where each node can have any number of children. 

# What is a binary tree? 

### A binary tree is one with at most 2 children connected to it. 

# Why did computer scientists create binary trees? 

### To make computation more efficient and algorithm faster 

# What are the advantages of limiting a node to two children? 

### A more faster computation 

# What is one disadvantage of that limitation? 

### Not all real world data can be represented using this binary tree.
