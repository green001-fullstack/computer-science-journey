# What Problem does a tree solve?

A tree is a hierarchical data structure used to represent parent-child relationships between pieces of data. In real life, data are not always linear
and there needs to be a way to represent the relationship between data in real life. A tree helps
to solve this problem by representing data in a non-linear format, thus helping to represent the relationship
and hierachy between them.

# Why arrays aren't enough for hierarchical data.

Arrays only gives a linear representation of data. It does not show the relationship between them.
